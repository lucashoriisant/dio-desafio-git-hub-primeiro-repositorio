Para clonar um projeto do ambiente remoto para o ambiente local, é necessário:

- Criar um novo repositório;
- Sempre detalhando com o máximo de detalhes do que se trata o projeto;
- Após esse processo na primeira aba do repositório criado no GitHub, selecione o ícone em verde (<> Code);
- Ao selecionar, escolha o clone do tipo HTTPS;
- Copie o link do repositório;
- Crie um diretório onde será armazenado o código localmente;
- Clique com o botão direito nesta página e selecione a opção 'Git Bash Here';
- Na pasta escolhida, insira o comando 'git clone' + (link adquirido no GitHub);
- Ao clonar o projeto, navegue na pasta do repositório 'cd' + <dir>.
- Digite 'git status' para verificar se está no diretório principal (main);
- Digite 'git add .' para adicionar o conteúdo no repositório local do Git;
- Digite 'git status' para verificar se houve preparação para "commitar";
- Digite 'git comit -m "(Nome do commit, certifique-se de digitar um comentário relevante)";
- Digite 'git status', para verificar se houve preparação para realizar o 'pull';
- Digite 'git push origin main', para enviar o repositório do Git local para o repositório do GitHub remoto;
